## ✨ Features

- React v17
- Tailwind.css v3 + styled-components (+ twin.macro)
- Useful atoms components

## 🚗 Run

```sh
yarn install
yarn dev
```

## 🔍 Check non-ascii characters

```sh
yarn ascii
```

## 📥 Update dependencies

```sh
python3 ./scripts/yarnd.py
```
